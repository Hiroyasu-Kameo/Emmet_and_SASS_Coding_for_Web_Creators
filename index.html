Emmetについて

EmmetとはHTMLやCSSなどを記述する際に単純な形式の記述を展開することによって素早く編集することができる、ソースコードエディタ用のプラグインです。
また、エディタによっては初めからEmmetが組み込まれているものもあります。

具体的な例を一つ紹介します。

エディタで次のコードを記述してEnter（エディタによってはTab）キーを押すと、

div#page<div+ul<li*5a

次のHTMLに展開されます。

<div id="page">
  <div></div>
  <ul>
    <li><a href=""></a><>li>
    <li><a href=""></a><>li>
    <li><a href=""></a><>li>
    <li><a href=""><>a><>li>
    <li><a href=""><>a><>li>
  </ul>
</div>

Sassについて

Sass(サース、サス)：Syntactically Awesome Style sheets（構文的に素晴らしいスタイルシート）とは、
CSSを効率的に書くことができるスタイルシート言語です。

Sassの発音は、設計者であるハンプトン・キャトリンさんご本人の動画を参照ください。The Future of Sass by Hampton Catlin

エディタについて

Emmetは様々なソースコードエディタにインストール出来るプラグインが用意されています。
Emmetプラグインのダウンロードサイト
Visual Studio Code（通称VSCode）には標準でEmmetが組み込まれています。
VSCodeは人気のあるテキストエディタの中で使い心地がよく筆者が愛用している、
というだけなので素晴らしいEmmet対応のソースコードエディタは他にもあります。
現在人気のあるEmmet対応のソースコードエディタを二つ挙げます。
リンク先からダウンロードすることが出来ます。

  • Visual Studio Code
  • Atom

また、伝統的なテキストエディタにはVim、そしてEmacsがありますが、
このどちらかのエディタを使いこなしておられる方は、このコンテンツで学ぶことは何もないでしょう。
