Emmetについて

EmmetとはHTMLやCSSなどを記述する際に単純な形式の記述を展開することによって素早く編集することができる、ソースコードエディタ用のプラグインです。
また、エディタによっては初めからEmmetが組み込まれているものもあります。
Emmet公式サイト

Emmetの具体的な例を二つ紹介します。

  • エディタのHTML編集モードで次のコードを記述してEnter（エディタによってはTab）キーを押すと、

    div#page>div+ul>li*5a

    次のHTMLに展開されます。 

    <div id="page">
  <div></div>
  <ul>
    <li><a href=""></a><>li>
    <li><a href=""></a><>li>
    <li><a href=""></a><>li>
    <li><a href=""><>a><>li>
    <li><a href=""><>a><>li>
  </ul>
</div>

  • エディタのHTML編集モードで次のコードを記述してEnter（エディタによってはTab）キーを押すと、

    !

    次のHTMLに展開されます。 

    <!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Document</title>
</head>
<body>

</body>
</html>

Sassについて

CSSを効率的に書くことができるCSS拡張メタ言語です。

Sassの発音は、設計者であるハンプトン・キャトリンさんご本人の動画を参照ください。
